!play <query> - Adds the song to the queue if it's found. Can be either a search query or a link
!join - Makes the bot join the same voice channel as the user
!disconnect - Makes the bot leave the voice channel
!queue - Displays the current songs in queue
!remove <position> - Removes the song specified by position from the queue
!pause - Pauses the current song playing
!resumes - Resumes the song if it's paused
!skip - Skips the current song playing
!shuffle - Shuffles the queue, making the order of the songs random
